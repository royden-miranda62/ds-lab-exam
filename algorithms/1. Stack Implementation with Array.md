# Question
Write a program to simulate the working of a stack using an array with the following operations:  
	a) Push  
	b) Pop  
	c) Display  
The program should print appropriate messages for stack overflow and stack underflow.


# Declarations
`#include <stdio.h>` : for printing to the console.  
`#include <stdbool.h>` : for handling Boolean values.
  
`#define SIZE 5` : sets the maximum size of the stack (as a constant).

`int stack[SIZE];` : declares an integer array as a stack with the defined maximum size.  
`int top = -1` : initializes the top pointer of the stack to -1.

> [!info] Note  
> A stack is full when the top pointer is one less than the size (`top = SIZE - 1`).  
> A stack is empty when the top pointer is -1.


# Functions
## 1. `void push(item)`
### Algorithm
- Step 1: Check if the stack is full.
	- if so, print an "overflow" statement and return control.
	- else, continue with the function.
- Step 2: Increment the top pointer stack by 1.
- Step 3: Assign `item` as the new top element.
- Step 4: Print an "pushed" statement.
### Code

``` c
void push(int value)
{
    if (isFull()) // top at max. index
    {
        printf("Stack overflow. Cannot push %d.\n", item);
        return;
    }

    stack[++top] = item; // incrementing the top before assignment
    printf("Pushed %d to stack.\n", item);
}
```

## 2. `void pop()`
### Algorithm
- Step 1: Check if the stack is empty.
	- if so, print an "underflow" statement and return control.
	- else, continue with the function.
- Step 2: Print the element at the top with a "popped" statement.
	- alternatively, assign it to a variable and then print it with a "popped" statement.
- Step 3: Decrement the top pointer by 1.
### Code

``` c
void pop()
{
    int popped;
    if (isEmpty()) // top below min. index
    {
        printf("Stack underflow. Cannot pop.\n");
        return;
    }

    popped = stack[top--]; // decrementing the top after assignment
    printf("Popped %d from stack.\n", popped);
}
```

## 3. `void peek()`
### Algorithm
- Step 1: Check if the stack is empty.
	- if so, print an "empty" statement and return control.
	- else, continue with the function.
- Step 2: Print the element at the top of the stack.
### Code

``` c
void peek()
{
    if (isEmpty())
        printf("Stack is empty.\n");
    else
        printf("Top element: %d\n", stack[top]);
}
```

## 4. `void display()`
### Algorithm
- Step 1: Check if the stack is empty.
	- if so, print an "empty" and return control.
	- else, continue with the function.
- Step 2: Loop through the stack from the top down (`top` to `-1`).
	- print all the elements one by one.
### Code

``` c
void display()
{
    if (isEmpty())
    {
        printf("\nStack is empty.\n");
        return;
    }

    printf("\nStack elements (top to bottom): \n");
    printf("------\n");
    for (int i = top; i > -1; i--)
        printf("| %d |\n", stack[i]);
    printf("------\n\n");
}
```


# Complete Program

``` c
#include <stdio.h>
#include <stdbool.h>

#define SIZE 5

int stack[SIZE];
int top = -1; // initializing the top pointer

// check if the stack is full
bool isFull()
{
    return (top == SIZE - 1);
}

// check if the stack is empty
bool isEmpty()
{
    return (top == -1);
}

// insert an element into the stack
void push(int item)
{
    if (isFull()) // top at max. index
    {
        printf("Stack overflow. Cannot push %d.\n", item);
        return;
    }

    stack[++top] = item; // incrementing the top before assignment
    printf("Pushed %d to stack.\n", item);
}

// delete an element from the stack
void pop()
{
    int popped;
    if (isEmpty()) // top below min. index
    {
        printf("Stack underflow. Cannot pop.\n");
        return;
    }

    popped = stack[top--]; // decrementing the top after assignment
    printf("Popped %d from stack.\n", popped);
}

// print the element at the top of the stack
void peek()
{
    if (isEmpty())
        printf("Stack is empty.\n");
    else
        printf("Top element: %d\n", stack[top]);
}

// print all the elements in the stack
void display()
{
    if (isEmpty())
    {
        printf("\nStack is empty.\n");
        return;
    }

    printf("\nStack elements (top to bottom): \n");
    printf("------\n");
    for (int i = top; i > -1; i--)
        printf("| %d |\n", stack[i]);
    printf("------\n\n");
}

int main()
{
    printf("----- Stack Operations -----\n");
    printf("Size of stack = %d\n", SIZE);

    printf("\n--- Push ---\n");
    push(10);
    push(20);
    push(30);
    push(40);
    push(50);
    push(60);

    display();

    printf("\n--- Pop ---\n");
    pop();
    pop();
    pop();

    display();

    pop();
    pop();
    pop();

    display();

    printf("\n--- Peek ---\n");
    push(1);
    push(2);
    push(3);
    peek();

    return 0;
}
```
