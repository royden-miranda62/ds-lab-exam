# Question
Write program to convert a given valid parenthesized infix arithmetic expression to a postfix expression. The expression consists of single character operands and the binary operators + (plus), - (minus), * (multiply) and / (divide).

---
# Declarations
`#include <stdio.h>`  
`#include <ctype.h>` : for the `isalnum()` function.  
`#include <string.h>` : for the `fgets()` function.

`#define SIZE 50` : sets the maximum length of the infix and postfix strings, and of the stack.

`char stack[SIZE];` : declares a character array as a stack.  
`int top = -1;`

---
# Functions
## 1. `void push(item)`
### Algorithm
- Step 1: Check if the stack is full.
	- if so, print an "overflow" statement and return control.
	- else, continue with the program.
- Step 2: Increment the top of the stack by 1.  
- Step 3: Assign `item` as the new top.

### Code

``` c
void push(char item)
{
    if (top >= SIZE - 1)
    {
        printf("Stack Overflow\n");
        return;
    }
    
    stack[++top] = item;
}
```

## 2. `char pop()`
### Algorithm
- Step 1: Check if the stack is empty.
	- if so, return the null character (`\0`).
	- else, continue with the function
- Step 2: Return the character at the top of the stack.
- Step 3: Decrement the top pointer by 1.
### Code

``` c
char pop()
{
    if (top == -1)
        return '\0';
    else
        return stack[top--];
}
```

## 3. `int precedence(symb)`
### Algorithm
- Step 1: If `symb` is a mathematical operator give it a precedence.
	- if `symb` is `^`, give it the highest precedence of 3.
	- if `symb` is `*` or `/`, give it a precedence of 2.
	- if `symb` is `+` or `-`, give it the lowest precedence of 1.
	- if `symb` is not an operator, return 0.
### Code

``` c
int precedence(char symb)
{
    if (symb == '^')
        return 3;
    else if (symb == '*' || symb == '/')
        return 2;
    else if (symb == '+' || symb == '-')
        return 1;
    else
        return 0;
}
```

## 4. `void convertToPostfix(infix[], postfix)[]`
### Algorithm
- Step 1: Initialize indices for the infix string (`i`) and postfix string (`j`).
- Step 2: Push '#' into the stack (sentinel character).
	- this acts as a stop signal for popping.
- Step 3: Loop over the infix string till the null terminator and evaluate each character (`ch`).
	- if `ch` is a space or a newline, continue on in the loop.
	- if `ch` is an alphabet or number, i.e. an operand, append it to the postfix string.
	- if `ch` is an opening parenthesis, push it to the stack.
	- if `ch` is a closing parenthesis:
		- pop each character from the stack and append it to the postfix string till met with an opening parenthesis or the sentinel character.
		- finally, pop the opening parenthesis and discard it.
	- if `ch` is an operator:
		- while the precedence of `ch` is lower than or equal to the precedence of the character at the top of the stack, pop the top character and append it to the postfix string.
		- loop over the stack till met with the sentinel character.
		- push `ch` onto the stack.
- Step 4: Loop over the stack till met with the sentinel character; pop all the remaining characters and append them to the postfix character.
- Step 5: Append the null terminator to the postfix string.
### Code

``` c
void convertToPostfix(char infix[], char postfix[])
{
    char ch;
    int i = 0, j = 0;

    push('#'); // "sentinel" character, avoids having to repeatedly checking for isEmpty

    while ((ch = infix[i++]) != '\0')
    {
        if (isspace(ch))
            continue;

        if (isalnum(ch)) // for variables/operands
            postfix[j++] = ch;

        else if (ch == '(')
            push(ch);

        else if (ch == ')')
        {
            while (stack[top] != '#' && stack[top] != '(')
                postfix[j++] = pop();
            pop(); // discards '('
        }

        else
        {
            while (stack[top] != '#' && precedence(ch) <= precedence(stack[top]))
                postfix[j++] = pop();
            push(ch);
        }
    }

    while (stack[top] != '#')
        postfix[j++] = pop();

    postfix[j] = '\0';
}
```

# Complete Program

``` c
#include <stdio.h>
#include <ctype.h>
#include <string.h>

#define SIZE 50

// declaring and initializing a stack
char stack[SIZE];
int top = -1;

// insert a character into the stack
void push(char item)
{
    if (top >= SIZE - 1)
    {
        printf("Stack Overflow\n");
        return;
    }

    stack[++top] = item;
}

// delete a character from the stack
char pop()
{
    if (top == -1)
        return '\0';
    else
        return stack[top--];
}

// check the precedence of a character (pedmas)
int precedence(char symb)
{
    if (symb == '^')
        return 3;
    else if (symb == '*' || symb == '/')
        return 2;
    else if (symb == '+' || symb == '-')
        return 1;
    else
        return 0;
}

// convert a given infix expression to postfix
void convertToPostfix(char infix[], char postfix[])
{
    char ch;
    int i = 0, j = 0;

    push('#'); // "sentinel" character, avoids having to repeatedly checking for isEmpty

    while ((ch = infix[i++]) != '\0')
    {
        if (isspace(ch))
            continue;

        if (isalnum(ch)) // for variables/operands
            postfix[j++] = ch;

        else if (ch == '(')
            push(ch);

        else if (ch == ')')
        {
            while (stack[top] != '#' && stack[top] != '(')
                postfix[j++] = pop();
            pop(); // discards '('
        }

        else
        {
            while (stack[top] != '#' && precedence(ch) <= precedence(stack[top]))
                postfix[j++] = pop();
            push(ch);
        }
    }

    while (stack[top] != '#')
        postfix[j++] = pop();

    postfix[j] = '\0';
}

int main()
{
    char infix[SIZE], postfix[SIZE];

    printf("Enter an infix expression: ");
    fgets(infix, SIZE, stdin);

    convertToPostfix(infix, postfix);
    printf("Postfix expression: %s\n", postfix);

    return 0;
}
```
